README

This is the specialized repository for Arxil, the language in EDSOS framework.
Arxil is EDSOS’s native intermediate language—designed as a faithful syntactic encoding of the AS model.
To see more about EDSOS framework, please visit [https:](https://github.com/ctysyh/EDSOS).
