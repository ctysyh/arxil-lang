# Arxil

[![License: Apache-2.0](https://img.shields.io/badge/License-Apache--2.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

This is the specialized repository for Arxil, the language in EDSOS framework.
Arxil is the native intermediate language of the [EDSOS](https://github.com/ctysyh/EDSOS) framework.
It provides a faithful syntactic encoding of the Arbor Strux (AS) model, serving as the bridge between high-level specifications and executable implementations.
